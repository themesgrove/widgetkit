== Description ==

WidgetKit for Elementor is a pack of  premium addons for Elementor. It is a free WordPress Plugin that extends Elementor Page Builder’s capability with 12 fully customizable elements.This plugin helps you build impressive websites without any sort of coding.
The ability to enable / disable elements comes with the plugin, so you can enable only the elements you needed for your website and ensure faster performance. 
WidgetKit for Elementor has been developed with clean code standard and meets W3C Validation. The latest CSS3, HTML5 and PHP 7.x is used in this plugin to give you a professional addon experience for Elementor Page Builder. WidgetKit for Elementor is WordPress 4.9.x ready.

WidgetKit for Elementor is useful for :

* Create any kind of complicated design without coding.
* Fully responsive layout for all devices
* Huge customization and easy to use.
* Well decoration settings options.
* Multiple feature on every block.
* Compatible with any kind of theme.
* User friendly structure.
* And much more…

== Features ==

* Slider with animation.
* Countdown.
* 4 style team block 
* Portfolio 3 style with filter 
* Custom blog 5 items
* Testimonial 3 styles
* 4 Differents pricing with tab
* Social share with animation 
* Custom carousel with post 
* Button with 10 hover animation 
* Modal with shortcode and video
* Feature hover effect with image and icon

== Installation ==

1. Upload `widgetkit-for-lementor.zip` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
